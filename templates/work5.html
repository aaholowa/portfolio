005 : JPG Converter

Description:This project is a simple but useful program that can convert any image type to JPEG format. The idea behind this project came from my coworkers who were struggling with manually converting images to JPEG format for uploading job site photos, which is our company's standard. I saw an opportunity to use my Python skills to save time and effort for my colleagues. The Image JPG Converter is a command-line tool that takes two arguments, the input folder, and the output folder, and converts all the images in the input folder to JPEG format and saves them in the output folder. The program uses the PIL library to open, convert, and save the images. The JPG Converter is a simple but effective tool that can save time and effort for anyone who needs to convert images to JPEG format. By creating this program, I have demonstrated my ability to identify a problem and use my Python skills to solve it. I believe that my passion for learning and my willingness to take initiative can be of value to any company that values innovation and efficiency.

Key Python Concepts:
Command-line arguments: The program takes two arguments from the command line, which are used as input and output folders.
File handling: The program uses the os module to check if the output folder exists and creates it if it doesn't. It also uses the os module to loop through the files in the input folder.
Image processing: The program uses the PIL library to open and save the images. It also uses the os module to split the file names and extensions.
