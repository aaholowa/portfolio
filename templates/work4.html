004 : Password Checker

Description:This Python project was designed to help users determine if their passwords have been involved in data breaches. By communicating with the URL from pwnedpasswords.com, this program allows users to input multiple passwords to check if any of them have been compromised. In this project, I learned about API's, URLs, and hashing passwords. Additionally, I was introduced to the concept of hash tables, which were used to store password information from pwnedpasswords.com. By using hash tables, the program can quickly search for password breaches without having to check every single password in the database. One of the main highlights of this project is that it helps users to increase their online security by checking if their passwords have been previously compromised. I also had a lot of fun developing this project, and it demonstrated my willingness to learn and my curiosity in exploring new technologies.

Key Python Concepts:
APIs: This project uses an API from pwnedpasswords.com to access data about compromised passwords. By utilizing an API, the program can easily access the data it needs to check if a password has been breached.
URL manipulation: The program creates a URL to access the API, allowing it to retrieve information about a specific password.
Hashing: To protect user passwords, the program hashes the input password and only sends the first five characters of the hash to the API. This ensures that the user's password is not transmitted in plain text.
Handling HTTP responses: The program handles HTTP responses to ensure that it can access the data it needs. If the response is not successful, the program raises an error.
Command-line arguments: Users can input multiple passwords to check from the command line, allowing them to quickly check their passwords without having to manually enter them one by one.
