001 : LCD Memory Game

Description: Collaborated within a group of 4 students to develop an engaging and challenging LCD memory game as our final project for an embedded systems course. The project is a memory game where players have to remember a sequence of numbers and repeat it accurately. We implemented the game using the ATmega328p microcontroller along with a keypad and LCD. The game starts by generating a random sequence of numbers on the LCD display, which the player has to memorize. The difficulty level increases as the game progresses, with longer and more complex sequences. The player has to input the correct sequence using the keypad to advance to the next level. If the player makes a mistake, they lose a life, and the game ends when all lives are lost. To add an element of excitement and urgency, we included a buzzer that sounds when the player loses a life. We also added a scorekeeping feature to keep track of the player's progress and display the score and lives remaining on the LCD. Our team implemented interrupt-based programming to handle the keypad input and the buzzer sound, making the game responsive and interactive. We also used various programming techniques, such as dynamic memory allocation, bitwise operations, and delay loops, to optimize the game's performance and reduce code complexity. Overall, our LCD memory game is a fun and challenging project that demonstrates our team's proficiency in embedded systems programming and hardware interfacing. We believe that this project showcases our creativity, technical skills, and teamwork, making us valuable candidates for embedded systems-related positions.
