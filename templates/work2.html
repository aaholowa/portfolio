002 : Python Suduko Solver

Description:The Sudoku Solver is a personal project I created to apply the Python programming skills I learned in an online course. The project required me to use my knowledge of the Python programming language, algorithmic thinking, problem-solving, debugging skills, attention to detail, and persistence. I wanted to challenge myself to create a useful tool that could solve Sudoku puzzles, which are known for being complex and difficult to solve. The Sudoku Solver is a command-line tool that allows users to input a Sudoku board and then solves it using a backtracking algorithm. The user inputs the board values through the command line, and the solver returns the solved board as an output. The solver uses a recursive algorithm to fill in each empty cell in the board with a valid number, ensuring that the board remains valid at each step. The solver first displays the unsolved board and then calls the solve() function to solve the board. The solve() function uses the empty() function to find the first empty cell in the board, and then uses the valid() function to determine the valid solutions for that cell. If a valid solution is found, the solver adds the number to the cell and calls the solve() function again on the updated board. If the solve() function returns True, the solver has successfully solved the Sudoku puzzle. The project not only helped me to further develop my programming skills, algorithmic thinking, and problem-solving abilities but also gave me a deeper understanding of the power of functional programming in solving complex problems. Overall, this project was a great way to apply the skills I learned in my online Python course and challenge myself to create something fun and useful.
